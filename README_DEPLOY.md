# Déploiement sur Tiiny.host

Voici les étapes pour mettre votre site en ligne :

1.  **Compresser le dossier** : Sur votre Mac, faites un clic droit sur le dossier `site boulange` et choisissez **Compresser "site boulange"**. Cela va créer un fichier `site boulange.zip`.
2.  **Aller sur Tiiny.host** : Ouvrez votre navigateur et allez sur [tiiny.host](https://tiiny.host).
3.  **Glisser-déposer** : Faites glisser le fichier `site boulange.zip` sur la zone de téléchargement du site.
4.  **Nommer le site** : Choisissez un nom pour votre sous-domaine (ex: `pain-et-passion`).
5.  **Publier** : Cliquez sur le bouton pour publier. Votre site sera en ligne instantanément !

---
*Note : Assurez-vous que le fichier index.html se trouve bien à la racine du ZIP pour que Tiiny.host le reconnaisse.*
